PLAY mode : 
  left clic to create an impenetrable field around the mouse 
  right clic to create an explosion, the longer you press, the bigger the explosion 
  press mouse wheel to concentrate all the particles in one point 
  press T to teleport all particles to a random location, spam it 3 or 4 times for a better looking result 
DRAW mode : 
  press B for Broken Lines
  press L for lines
  press C for circles
  press P for free-hand drawing
